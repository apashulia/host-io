Instruction how to run server and aplication

step 1: unzip file

step 2: go to backend folder and run the command 'npm i' to install all dependencies with the help of any terminal you want

step 3: after step 2 run the command 'npm run dev' to start server localy

step 4: open browser and go to 'https://sprightly-sopapillas-364652.netlify.app/'
